# Configuration

## Environment Variables

| Variable          | Default | Description                     |
| ----------------- | ------- | ------------------------------- |
| `PORT`            | `4000`  | Server port                     |
| `LOG_BUFFER_SIZE` | `1000`  | Max logs in memory              |
| `API_KEY`         | -       | Optional API key for authentication |
| `CORS_ORIGIN`     | `*`     | CORS allowed origins            |
